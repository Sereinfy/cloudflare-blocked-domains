# 🌐 Cloudflare Blocked Domains List

### 📖 简介

在日常使用中，我们经常会遇到这样的问题：访问某些网站（如 `nexcess.net`）时，页面提示  
> **Sorry, you have been blocked**

这通常意味着目标站点 **无法通过直连正常访问**，必须通过 **国外节点** 才能打开。

本项目的目标是提供一份由社区共同维护的 **Cloudflare 被拦截/受限域名列表**，以便：

- ✅ 自动将这些网站分流到国外节点  
- ✅ 减少用户手动维护域名分流规则的频率  
- ✅ 提升访问稳定性与速度体验  

---

### ⚙️ 背景说明

许多网站使用了 **Cloudflare CDN 服务**，但情况并不完全一致：

1. **部分网站在直连时可正常访问**  
   - 例如启用了 Cloudflare 的加速，但未做地域限制。  
   - 对于这类网站，直连通常能获得更快的访问速度。

2. **部分网站存在访问限制或被墙**  
   - 包括 Cloudflare 主动拦截（返回 403 / “You have been blocked”），或源站被防火墙屏蔽。  
   - 这些网站必须通过国外节点访问。  

对于使用 **mosdns 进行 IP 优选** 的用户来说，默认通常是 **Cloudflare 域名直连**。  
但遇到上述“被拦截”网站时，就需要手动将其加入国外分流列表，操作繁琐。  

因此，本项目旨在通过 **社区协作**，维护一份共享的被拦截域名清单，让每位用户都能从中受益。

---

### 🧩 适用场景

- 使用 **mosdns** 进行 IP 优选的用户  
- 希望 **Cloudflare 优化直连 + 自动代理分流** 的场景
